Architecture

SOTA Architecture

The SOTA System includes a client and server, communicating using the RVI protocol. The SOTA Server is comprised of three sub-modules:

  • core - The central component of the SOTA server that implements package queuing and distribution

  • external-resolver

    • An implementation of a package resolution server, which maps VINs to a list of software components to be installed on a vehicle

  • web-server

    • The web interface to the SOTA server, allowing administration of vehicles and software components
