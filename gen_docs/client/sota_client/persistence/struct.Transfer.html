pub struct Transfer {
    pub package: PackageId,
    pub checksum: String,
    pub transferred_chunks: Vec<u64>,
    pub prefix_dir: String,
    pub last_chunk_received: i64,
}

Type for storing the metadata of a in-progress transfer, which is defined as one package. Will clear out the chunks on disk when freed.

Fields

package

PackageId of this transfer.

checksum

SHA1 checksum of the fully assembled package.

transferred_chunks

Vector of transferred chunks.

prefix_dir

Path to the directory, where chunks will be cached and finished packages will be stored.

last_chunk_received

Timestamp, when the last chunk was received. Given as a unix epoch timestamp.

Read a chunk file file and append it to a package file. Returns a String with a error message should something go wrong.

Arguments

  • path: Pointer to a PathBuf where the chunks are cached.
  • index: Index of the chunk to append.
  • file: Pointer to a File where the chunk should be appended. Should be created with OpenOptions and the append only option. See the documentation for OpenOptions, File, and the implementation of assemble_chunks for details.

fn checksum(&self) -> bool

Verify the checksum of this transfer. Assumes the package was already assembled. Prints a error message showing the mismatched checksums and returns false on errors.
