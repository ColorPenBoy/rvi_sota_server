sota_client

Module sota_client::handler [−] [src]

Implements message handling for the rvi module.

Implements individual message handler for the different messages the client might receive, and the general ServiceHandler that parses messages and delegates them to the individual handlers.

This is a reference implementation for the rvi module, that can later be split out into a crate.
